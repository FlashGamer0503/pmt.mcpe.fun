Where do I start?

This website has many tools such as ZIP Converting, PHAR Converting and API Modification. These tools are avalible for you to use free at charge. That means you will not need to install DevTools when you want to convert a plugin. The best place to start is to use the PHAR Compiler to make your plugins compact. You can click on one of the left links to begin.
