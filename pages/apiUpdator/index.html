What's an API changing?

On PocketMine, API changes marks a new era on plugin API of pocketmine where some plugin would basicly break
But most plugin would just work fine on your server. Thought they may be less stable
We provide some incompatibility changes with the new apis to try making your plugin working.
Thought note that neither us, the plugin author, or the PocketMine team will be responsible for any damage done to your server or your machine using this convertor.

How do I force API change?

  • Download your incompatible plugin in phar form, of course.
  • Make sure everything is the way you want it.
  • Select needed changes to correct some incompatibilities
  • Put the file somewhere you can find it, then click on the paperclip and select "Convert"
